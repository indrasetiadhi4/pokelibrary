# PokeLibrary

> Author: Indra Setiadhi P.

## Installation

```bash
# Clone the repository
$ git clone https://github.com/indrasetiadhi4/pokelibrary.git

# Change directory
$ cd pokelibrary

# Install Dependencies
$ npm install

# Run app
$ npm run dev
```
